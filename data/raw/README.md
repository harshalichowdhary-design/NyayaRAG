# Raw Dataset

This directory contains the original Indian Supreme Court judgment dataset used in the NYAYaRAG project.

## Dataset

Source: Open Indian Supreme Court Judgments Dataset
Coverage: Judgments delivered in 2024
Number of judgments: 782
Format: PDF

The files in this directory are maintained in their original form without modification.

Processed datasets, chunked corpora, embeddings, and evaluation files will be added in future stages of the project.
